Overview  
This project is an interactive Excel Dashboard that analyzes **bike sales data**.  
It provides insights into sales trends, customer demographics, and product performance.  

Features  
- Interactive **slicers & filters** for region, age group, and gender  
- KPIs showing **total sales, average income, and commute patterns**  
- Charts and visuals for easy decision-making  
- Clean and professional dashboard layout  

 Tools Used  
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Slicers and Conditional Formatting
